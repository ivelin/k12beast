# Parents Guide

Support your child’s learning journey with K12Beast’s personalized tutoring tools.

## Getting Started

1. **Sign Up**: Create an account at [/public/signup](/public/signup).
2. **Start a Session**: Submit a problem at [/chat](/chat) for your child.
3. **Review Progress**: Monitor shared sessions to track improvement.
