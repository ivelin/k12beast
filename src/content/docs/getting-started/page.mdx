# Getting Started

Welcome to K12Beast! This guide helps you begin using the app for K-12 education.

## First Steps

1. **Sign Up**: Create an account at [/signup](/signup).
2. **Explore**: Check out the documentation for your role (Parent, Student, Teacher, Tutor).
3. **Start Learning**: Submit a problem to begin a tutoring session.
