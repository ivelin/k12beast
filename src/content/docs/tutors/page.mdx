# Tutors Guide

Welcome to K12Beast, a personalized tutoring app designed to enhance K-12 education. This guide helps tutors understand how to use K12Beast to assist students in their learning journey.

## Getting Started

1. **Sign Up**: Create an account to get started.
2. **Start a Session**: Begin a session by submitting a problem for the student.
3. **Track Progress**: Monitor student improvement through session details.

## Key Features

- **Personalized Lessons**: Deliver lessons tailored to each student's needs.
- **Interactive Quizzes**: Use AI-generated quizzes to test student understanding.
- **Session Sharing**: Share session outcomes with students or parents for review.
