# Tutoring Guide for Tutors

This guide helps tutors leverage K12Beast to provide effective, personalized tutoring for K12 students.

## Getting Started

1. **Sign Up as a Tutor**: Create an account at `/signup` to access tutoring features.
2. **Start a Session**: Initiate a new session at `/chat` to create tailored lessons.
3. **Track Student Progress**: Review session details and quiz results on the `/history` page.

## Best Practices

- **Customize Lessons**: Use student-submitted problems to generate AI-driven lessons via the chat interface.
- **Incorporate Quizzes**: Request quizzes to reinforce concepts and assess understanding.
- **Engage Parents**: Share session links with parents to keep them updated on student progress.

For more information, see the [Tutors Overview](/public/docs/tutors).
